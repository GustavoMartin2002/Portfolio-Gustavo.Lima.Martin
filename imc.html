Calculadora IMC


LEIA-ME

Este projeto foi desenvolvido em Python utilizando a biblioteca Tkinter para criar uma interface gráfica simples.
Ele permite ao usuário inserir sua altura e peso, para calcular o Índice de Massa Corporal (IMC).


Como funciona?

  • O usuário insere sua altura (em metros) e peso (em quilogramas).
  • O programa calcula automaticamente o IMC com base nesses valores.
  • O resultado é exibido na tela com a classificação do IMC (abaixo do peso, peso normal, sobrepeso, etc).

    • Tecnologias utilizadas:

  • Python: Linguagem de programação principal.
  • Tkinter: Biblioteca para criação da interface gráfica.
